Quick and basic example of working with git

Here is the workflow I attempted to demonstrate during the Git Rganized workshop.

1. Create a repository on Git Hub.

i) Log in to GitHub.

ii) Create a new repository.

Click New repository from the + dropdown menu on the top-right of the welcome screen next to your profile pic.

Fill in the fields of the Create a new repository page.

Choose whether you want your repos to be public or private (you can make private repositories public later). I recommend you click Initialize this repository with a README so you wont forget to create a README later. I also recommend you add a .gitignore for the scripting language that you use. Add a license to allow other people to use your work.
